# Paquete documental ANKLO — versión 1.0

Este paquete contiene tres documentos coordinados:

1. **Manual Maestro de Supervisión de Anclajes**: guía operativa, técnica, normativa, de seguridad, calidad y gestión.
2. **Resumen Maestro del Proyecto**: contexto sistematizado para conservar continuidad entre fases y futuras conversaciones.
3. **Bosquejo Funcional y Arquitectura del ERP ANKLO-OS**: alcance, módulos, datos, tecnología, seguridad, modo offline y hoja de ruta.

## Formatos

Cada documento se entrega en Markdown editable y PDF. Los archivos SVG de la carpeta `assets` forman parte de los documentos Markdown; mantenga la estructura de carpetas al copiarlos o versionarlos.

## Alcance de validación

La versión 1.0 establece una base técnica verificable. Antes de convertirla en procedimiento corporativo aprobado deben incorporarse los productos realmente utilizados, sus instrucciones de instalación vigentes, planos y especificaciones contractuales, formatos internos de ANKLO, matriz de riesgos por tarea y criterios de aceptación aprobados por el responsable estructural y de seguridad.

## Control de cambios recomendado

Use control de versiones y registre: versión, fecha, autor/revisor, secciones modificadas, motivo y aprobación. No reemplace silenciosamente una instrucción de fabricante o un criterio de aceptación.
