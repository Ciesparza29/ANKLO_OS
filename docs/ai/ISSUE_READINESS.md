# Preparación de Issues (ISSUE READINESS)

## Estados del Issue

Un agente puede proponer el estado, y de ser así, **Israel lo aprueba**. Ningún estado autoriza automáticamente escritura.

- `NO_LISTO`: faltan datos, autoridad, fuentes o aceptación.
- `LISTO_PARA_PLAN`: existe información suficiente para diseñar un plan, pero no para implementar.
- `LISTO_PARA_IMPLEMENTACION`: el plan fue revisado y aprobado, pero la ejecución todavía requiere autorización expresa de Israel.
- `BLOQUEADO`: existe contradicción, riesgo, dependencia o falta de evidencia que impide avanzar.
