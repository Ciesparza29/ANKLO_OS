# Registro de Habilidades de Terceros (THIRD PARTY SKILLS)

No existen Skills de terceros aprobadas o instaladas por este registro.

## Estados de Skills

- `CANDIDATA`: propuesta para revisión.
- `EN_AUDITORIA`: siendo inspeccionada por seguridad y compatibilidad.
- `APROBADA_NO_INSTALADA`: autorizada pero pendiente de instalación técnica.
- `INSTALADA`: activa y lista para uso por los agentes.
- `SUSPENDIDA`: temporalmente bloqueada.
- `RETIRADA`: eliminada permanentemente.
- `RECHAZADA`: evaluada y denegada.
